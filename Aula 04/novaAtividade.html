O que é HTML

Por: Lucian Bellini

Data: 14/03/2024

HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem padrão utilizada para criar e estruturar conteúdo na web. Desenvolvida para facilitar a formatação e organização de documentos para exibição em navegadores da internet, o HTML utiliza uma série de elementos e tags para definir a estrutura e o conteúdo de uma página da web.
